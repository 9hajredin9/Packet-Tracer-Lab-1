# Network Project/Lab

## Overview
Cisco Packet Tracer project demonstrating an enterprise network design with core routing and switching services.

## Features
- VLANs  
- Inter-VLAN Routing (Router-on-a-Stick)  
- OSPF Dynamic Routing  
- DHCP  
- DHCP Relay Agent  
- Static IP Addressing  

## Tools Used
- Cisco Packet Tracer

Topology diagram included in this repository (`Screenshot1.PNG`).

This project was created in Cisco Packet Tracer to demonstrate networking skills.
